# 盖软件学院院章流程

几乎所有文件只有盖了院章才有效果，包括报销单，审批表，集训证明等。

部分文件需要盖教务处的章，如证书等。

盖软件学院院章步骤：
- 将要盖章的文件电子版发送给[宋友老师](mailto:songyou@buaa.edu.cn)并约定时间
- 将要盖章的文件打印，于约定时间交给宋友老师（东301-2）签字
- 将签字后的文件交给王丽华老师（东309）盖章
- **请全程保持耐心和礼貌**


# 盖教务处章流程
